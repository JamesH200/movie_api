API Documentation

Endpoints Overview

Request URL Method Description Query Parameters Request Body Response
Get welcome message / GET Returns a welcome message None None Text string: "Welcome to my Top 10 Movies!"
Get all movies /movies GET Returns data on all movies None None JSON array of movie objects
Get documentation /documentation GET Serves the documentation HTML file None None HTML content of the documentation page
Get Movie by Title /movies/:title GET Returns data about a single movie specified by its title. title (URL parameter) None JSON object with movie details (title, description, genre, director)
Get Genre by Name /genres/:name GET Returns data about a genre specified by its name. name (URL parameter) None JSON object with genre details (name, description)
Get Director by Name /directors/:name GET Returns data about a director specified by their name. name (URL parameter) None JSON object with director details (name, bio, birth year, death year)
Register User /users/register POST Allows a new user to register. None JSON object with user data (username, password, email, dateOfBirth) Confirmation message or error message
Update User /users/:username PUT Allows a user to update their personal information. username (URL parameter) JSON object with updated user data (username, password, email, dateOfBirth) Updated user data or confirmation message
Add Movie to Favorites /users/:username/favorites POST Add a movie to a user's list of favorites. username (URL parameter) JSON object with movie title Updated list of favorite movies or confirmation message
Remove Movie from Favorites /users/:username/favorites/:movieTitle DELETE Remove a movie from a user's list of favorites. username (URL parameter), movieTitle (URL parameter) None Updated list of favorite movies or confirmation message
Deregister User /users/:username DELETE Allows an existing user to deregister. username (URL parameter) None Confirmation message or error message
