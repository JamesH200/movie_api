API Documentation

Endpoints Overview

Request URL Method Description Query Parameters Request Body Response
Get welcome message / GET Returns a welcome message None None Text string: "Welcome to my Top 10 Movies!"
Get all movies /movies GET Returns data on all movies None None JSON array of movie objects
Get documentation /documentation GET Serves the documentation HTML file None None HTML content of the documentation page
Get All Movies /movies Returns a list of all movies in the database. GET None None JSON array of movie objects
Get Movie by Title /movies/:title Returns data about a single movie specified by its title. GET title (URL parameter) None JSON object with movie details (title, description, genre, director, image URL, etc.)
Get Genre by Name /genres/:name Returns data about a genre specified by its name. GET name (URL parameter) None JSON object with genre details (name, description)
Get Director by Name /directors/:name Returns data about a director specified by their name. GET name (URL parameter) None JSON object with director details (name, bio, birth year, death year)
Register User /users/register Allows a new user to register. POST None JSON object with user data (username, password, email, dateOfBirth) Confirmation message or error message
Update User /users/:username Allows a user to update their personal information. PUT username (URL parameter) JSON object with updated user data (username, password, email, dateOfBirth) Updated user data or confirmation message
Add Movie to Favorites /users/:username/favorites Add a movie to a user's list of favorites. POST username (URL parameter) JSON object with movie title Updated list of favorite movies or confirmation message
Remove Movie from Favorites /users/:username/favorites/:movieTitle Remove a movie from a user's list of favorites. DELETE username (URL parameter), movieTitle (URL parameter) None Updated list of favorite movies or confirmation message
Deregister User /users/:username Allows an existing user to deregister. DELETE username (URL parameter) None Confirmation message or error message
