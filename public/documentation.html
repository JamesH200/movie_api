Movie API

This API provides access to information about various movies, allowing users to retrieve details such as titles, genres, and more.

Endpoints

1. GET /movies

Retrieves a list of all movies.


    GET /movies
    Response:
    Status 200 OK
    Content-Type: application/json
    [
      {
        "title": "Movie Title",
        "description": "Movie Description",
        "director": "Movie Director",
        "genre": "Movie Genre",
        "featured": false
      },
      ...
    ]

2. GET /movies/:title

Retrieves data about a single movie by its title.


    GET /movies/Inception
    Response:
    Status 200 OK
    Content-Type: application/json
    {
      "title": "Inception",
      "description": "A thief who enters the dreams of others in order to steal information is tasked with planting an idea in a target's subconscious.",
      "director": "Christopher Nolan",
      "genre": "Sci-Fi, Action",
      "featured": true
    }

3. Put /movies

Adds a new movie to the collection.


    Put/movies
    Request Body:
    Content-Type: application/json
    {
        "title": "Inception",
        "description":
        "Movie Description.",
        "director": "Christopher Nolan",
        "genre: "Action, Sci-Fi",
        featured: true,
    }
    Response:
    Status 201 Created
    Content-Type: application/json
    {
      "title": "New Movie Title",
      "director": "New Director",
      "genre": "Genre",
      "description": "Movie description.",
      "featured": true
    }

4. DELETE /movies/:title

Deletes a movie by its title.


    DELETE /movies/Oblivion
    Response:
    Status 200 OK
    Content-Type: text/plain
    Movie titled "Oblivion" was deleted.

5. PUT /movies/:title/genre

Updates the genre of a movie by its title.


    PUT /movies/Inception/genre
    Request Body:
    Content-Type: application/json
    {
      "genre": "Action, Adventure"
    }
    Response:
    Status 200 OK
    Content-Type: application/json
    {
      "title": "Inception",
      "description": "A thief who enters the dreams of others in order to steal information is tasked with planting an idea in a target's subconscious.",
      "director": "Christopher Nolan",
      "genre": "Action, Adventure",
      "featured": true
    }

6. POST /movies

Adds a new movie to the collection.


    POST /movies
    Request Body:
    Content-Type: application/json
    { 
        "title": "21 Bridges",
        "director": "Brian Kirk",
        "genre": "Action, Crime, Thriller",
        "description": "An NYPD detective is thrust into a citywide manhunt for a pair of cop killers after uncovering a massive conspiracy."
        "featured": false
    }
    Response:
    Status 201 Created
    Content-Type: application/json
    {
      "title": "21 Bridges",
      "director": "Brian Kirk",
      "genre": "Action, Crime, Thriller",
      "description": "Movie description.",
      "featured": false
    }

7. GET /movies/genre/:genre

Retrieves movies filtered by genre.


    GET /movies/genre/Sci-Fi
    Response:
    Status 200 OK
    Content-Type: application/json
    [
      {
        "title": "Prometheus",
        "description": "A team of explorers discover a clue to the origins of mankind on Earth, leading them to an alien world.",
        "director": "Ridley Scott",
        "genre": "Sci-Fi, Horror",
        "featured": false
      },
      {
        "title": "Blade Runner 2049",
        "description": "A young blade runner's discovery of a long-buried secret leads him to track down former blade runner Rick Deckard.",
        "director": "Denis Villeneuve",
        "genre": "Sci-Fi, Action",
        "featured": true
      },
      ...
    ]

8. GET /movies/director/:director

Retrieves movies filtered by director.


    GET /movies/director/Christopher Nolan
    Response:
    Status 200 OK
    Content-Type: application/json
    [
      {
        "title": "Inception",
        "description": "A thief who enters the dreams of others in order to steal information is tasked with planting an idea in a target's subconscious.",
        "director": "Christopher Nolan",
        "genre": "Sci-Fi, Action",
        "featured": true
      }
    ]

9. Post /movies/:title/description

Updates the description of a movie by its title.


    Post /movies/Friday After Next/description
    Request Body:
    Content-Type: application/json
    {
      "description": "Updated description for Friday After Next."
    }
    Response:
    Status 200 OK
    Content-Type: application/json
    {
      "title": "Friday After Next",
      "description": "Updated description for Friday After Next.",
      "director": " Marcus Raboy",
      "genre": "Comedy, Drama",
      "featured": false
    }
